Generate Barcode

The Barcode Generation feature allows users to create barcodes for products. Supported barcode formats include Code 39, Code 128, and QR Codes.

Required Excel Structure

  • Column A: Product Name
  • Column B: Product Code (Unique Identifier)

Steps to Generate Barcodes

  1. Open the Barcode Creation module in PazinoSuite.
  2. Upload an Excel file containing product details.
  3. Provide Product name Column Value
  4. Provide Product Code Coulumn Value
  5. Provide Starting and End Row
  6. Click the 'Process' button.
  7. Select folder for Barcode images
  8. The software will generate barcodes and save them in the given folder.
